📈 Trading Games v1

Trade a contract whose value is a secret arithmetic result of the drawn assets.

Each player is privately dealt this many extra assets, never revealed to others. They count toward the main contract settlement just like the community cards. Note: hints describe the shared community cards only — your private card is extra and is not counted in them. 0 = off (everyone sees the same info).

Round advances automatically after 60 seconds. Set to 0 for manual-only.

Computer traders that play either mode. Each estimates fair value by simulation and quotes wider when unsure. In market-making mode they bid, make markets, and take (~15% of bids deliberately off); in open-outcry mode they post two-sided limit orders and pick off mispriced resting orders.
